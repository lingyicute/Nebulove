### 「Nebulove」？

「Nebulove」是星光与心事相拥时，诞生的名字：
**Nebula**（星云）揉碎深蓝夜幕里银河的柔光，把 **Love**（爱意）藏进圆润的转角。

想让它们替我带去比银河系更漫长的思念，
以及所有还没来得及说出口的「我好想你」。
